### 🏠 [RentMate - Full Stack Rental Platform](https://github.com/Cecil-B-Liv/RentMate)
A comprehensive rental management platform built with Spring Boot
- **Backend**: Spring Boot, RESTful APIs, JPA/Hibernate
- **Frontend**: React, Vite, Tailwind CSS
- **Database**: MySQL with complex relational design
- **Features**: Property management, lease processing, payment tracking, real-time chat

### 🌐 [RushNet - Social Media Platform](https://github.com/Cecil-B-Liv/fsDev)
A modern social media platform built with MERN stack
- **Backend**: Node.js, Express.js, MongoDB with session management
- **Frontend**: React, Bootstrap, responsive design
- **Features**: User authentication, posts, groups, notifications, friend system
- RESTful API design with middleware architecture
- Real-time notifications and search functionality

### 🎮 [CHIP-8 Emulator](https://github.com/Cecil-B-Liv/Chip8-Simulator)
A fully functional CHIP-8 emulator built in C using SDL3
- Implements complete CHIP-8 instruction set
- 60Hz timer system for accurate game timing
- Audio support with beep functionality
- Cross-platform compatibility

### 🤖 [Game AI Techniques](https://github.com/Cecil-B-Liv/Game-Technique)
Implementation of AI steering behaviors for game development
- Seek behavior with smooth movement physics
- Vector mathematics and physics simulation
- Real-time agent movement using Pygame
- Has 2 projects game in this repository: Snake eating - Shooting game

### 🌾 [Wheat Classification - ML Project](https://github.com/Cecil-B-Liv/WheatClassification)
Machine learning project for agricultural classification
- **Technology**: Jupyter Notebook, Python, scikit-learn
- **Purpose**: Machine Learning course assignment
- **Focus**: Classification algorithms for wheat variety identification
- Data preprocessing, feature engineering, and model evaluation
- Agricultural data science and pattern recognition

### 🔧 [Raspberry Pi Bare Metal OS](https://github.com/Cecil-B-Liv/Embedded3-Group)
Low-level operating system development for ARM64 architecture
- Written in C and ARM Assembly
